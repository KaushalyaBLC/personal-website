Parking Management Application

The Parking Management Application is a command-line utility designed to streamline vehicle entry and exit management in parking lots. As a software solution, it simplifies the process of recording vehicle details when entering the parking lot, tracks entry time, and calculates parking fees based on the duration of stay and vehicle type.

Key Features:

  • User-friendly interface: The application provides a simple and intuitive command-line interface for easy interaction.
  • Efficient fee calculation: The utility accurately calculates parking fees for vehicles based on their type and duration of stay.
  • Data integrity: The application ensures data integrity and efficient management of parking records.

How to Use:

  1. Enter Vehicle Details: Users can log vehicle entry by providing the vehicle number and selecting the vehicle type (Car or Motorcycle).
  2. Exit and Fee Calculation: When a vehicle exits the parking lot, the user enters the vehicle number. The application calculates the parking fee and displays the total fee.

Future Enhancements:

The application is open to future enhancements, including support for additional vehicle types, graphical user interface (GUI) implementation, and advanced features such as parking reservations and online payment options.

Technology Used:

The Parking Management Application is developed using the C programming language with minimal hardware and software requirements, making it a lightweight and efficient solution.
